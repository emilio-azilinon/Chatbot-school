# Chatbot scolaire - Prototype

Prototype de bot web pour université / service scolarité.

## Installation PHP avec WAMP

1. Vérifie que le dossier est dans `C:\wamp64\www\projet\Chatbot-school`
2. Lance WAMP
3. Ouvre le navigateur sur :

```
http://localhost/projet/Chatbot-school/
```

4. Le chat PHP utilise :
   - `index.php`
   - `chat.php`
   - `public/style.css`
   - `public/php-app.js`

## Fonctionnalités

- Backend Node.js + Express
- Frontend web simple (chat)
- Réponses fixes sur :
  - inscriptions
  - filières
  - contact / localisation
  - documents nécessaires
